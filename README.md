# Jober
> Bot Multitarefas.

<p>Um Bot com capacidade de executar uma infinidade de ações predefinidas tais como</p>
<ul>
<li>movimentação do mouse</li>
<li>manipulação do teclado</li>
<li>leitura de caracteres</li>
</ul>

![](share/Jober_ScreenShot.png)

## Instalação

<h3>Linux:</h3>
<p>make</p>
<p>make install</p>
